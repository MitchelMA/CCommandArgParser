notation_s structure initializer function.

Parameters
notationA pointer to the notation structure that's going to be initialized.
main_nameThe main name associated with this notation structure.
alias_nThe length of the va-list aliases. Value is allowed to be 0.
aliasesA va-list of const char* items with a length of alias_n.
Returns
false when notation or main_name is NULL.
false when main_name is invalid as described by notation_is_valid_flag().
false on any allocation failure, be it a call to malloc() or strdup().
true when none of the aforementioned apply.
