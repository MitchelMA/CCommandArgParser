Detailed Description

This is the command structure. This structure is used to indicate the expected logic of the application by the caller/callee. It holds an array of options denoted by: option_capacity, option_count, and the options array. The parameters that were used when calling the command are stored in parsed_arguments.

For functionality and usage of this structure, look into the command.h header-file.
